About the App

ProphecyML is a mobile application that uses machine learning to help educate users about machine learning using the stock market. We use Google Authentication to securely authenticate users and personalize their experience.

We Use   Google OAuth i OAuth 2.0 Authorization Code Flow with PKCE via System Browser (MSAL iOS), federated through Azure AD B2C using Google as an external identity provider. forauthentication2

We currently Support Linear-Regression based algorithms but soon plan to add more advanced models like Mamba (a new deep learning architecture based on Structured State Space Models (SSMs)), and LTSM.
